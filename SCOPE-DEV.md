# Scope Development

## In scope

- Repository foundation dan Infrastructure/Foundation.
- Data Ingestion & Integration (DI&I), Asset Repository, CMDB.
- Analytics & AI advisory, Workflow Automation draft/dry-run, SIEM/SOAR boundary.
- NOC-oriented Web Dashboard/API.
- Future Hermes read-only shadow integration setelah gate.

## Explicit exclusions Phase 0

- Deployment application components atau stack.
- Koneksi source Production, live ingestion, atau real credential.
- Autonomous remediation dan direct device/OT action.
- Production HA/DR, SLA, hardening claim, atau 24x7 operation.
- Demo memakai real Production data.
- Integrasi Hermes aktif, akses database, tool, credential, endpoint, atau operational evidence untuk Hermes.

Phase 0 hanya membangun policy, template, synthetic fixtures, local validation, CI guardrails, dan Dev Entry evidence.
